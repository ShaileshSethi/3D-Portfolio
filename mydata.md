# Shailesh Sethi Portfolio

This is a single-page personal portfolio built with Vite, React, TypeScript, Tailwind CSS, Lucide React, and Motion. The app is designed as a cinematic scroll experience with animated sections for the hero, about, projects, achievements, skills, contact, and footer.

## Local Development

### Prerequisites

- Node.js LTS

### Install Dependencies

```bash
npm install
```

### Run Locally

```bash
npm run dev
```

### Build For Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Notes For Updating The App

- Replace any placeholder or temporary links with your final public URLs
- Keep project cards short and outcome-focused
- Keep the one-page portfolio flow intact so the section anchors continue to work
- Update the email, LinkedIn, GitHub, and footer links together so the profile stays consistent
